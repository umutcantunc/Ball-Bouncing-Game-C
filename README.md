# Ball Bouncing Game

Raylib kütüphanesi ve C dili kullanılarak geliştirilmiş fizik tabanlı top sektirme oyunu.

## Özellikler
- Gerçekçi yerçekimi fiziği
- Mouse ile topa yön verme
- Sonsuz oyun döngüsü
- Tam ekran modu (F11)

## Nasıl Oynanır?
1. Topa tıklayarak havada tutmaya çalışın.
2. Mouse'un konumuna göre top farklı yönlere seker.
3. Yere düşürmemeye çalışın!

## Kullanılan Teknolojiler
- C Dili
- Raylib
- Code::Blocks
